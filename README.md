# Android Kotlin MVVM
This is a demo application for Kotlin, ACC, MVVM-Pattern, etc.
## Libraries
- Kotlin
- ACC(ViewModel, LiveData, Room)
- MVVM(Model-View-ViewModel) Pattern
- Databinding
- Retrofit2
- Coroutine
- Koin
- Glide
